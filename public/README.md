# Internet Safety - Образовательный сайт о безопасности в интернете

## Описание проекта

Современный одностраничный адаптивный сайт на тему интернет-безопасности, созданный для школьного проекта. 
Сайт рассказывает о видах интернет-мошенничества и способах защиты от них.

## Структура проекта

```
internet-safety-site/
├── index.html          # Главная HTML страница
├── styles.css          # Все стили CSS
├── script.js           # JavaScript для интерактивности
└── README.md           # Документация
```

## Технологии

- **HTML5** - семантическая разметка
- **CSS3** - стилизация с использованием Flexbox и Grid
- **JavaScript** - интерактивность и анимации
- **Google Fonts** - шрифты Montserrat и Open Sans

## Особенности

### Дизайн
- Современный минималистичный дизайн
- Цветовая палитра: #2F80ED (синий), #27AE60 (зелёный), #EB5757 (красный)
- Адаптивная сетка 12 колонок
- Скругленные карточки с тенями
- Плавные переходы и анимации

### Структура сайта

1. **Header** - Фиксированная шапка с навигацией
2. **Hero Section** - Главный экран с градиентом и иконками
3. **About Section** - Что такое интернет-мошенничество (3 карточки)
4. **Types Section** - Виды мошенничества (4 карточки)
5. **Signs Section** - Признаки мошенников (5 предупреждений)
6. **Protection Section** - Советы по защите (4 карточки)
7. **Footer** - Футер с информацией и навигацией

### Адаптивность

- **Desktop** (>768px): полная версия с горизонтальным меню
- **Mobile** (<768px): бургер-меню, одноколоночная сетка карточек

## Как использовать

1. Скачайте все файлы в одну папку
2. Откройте `index.html` в браузере
3. Наслаждайтесь современным адаптивным сайтом!
